## 1. Project Goal

The goal of this assignment is to build and deploy a static personal blog website. The website should be accessible through a public URL, managed with Git, and integrated with the previous homework content.

## 2. Tools and Platform

This project uses the following tools:

- HTML for webpage structure
- CSS for basic styling
- Markdown for documentation
- Git for version control
- GitHub Pages for website deployment

I chose GitHub Pages because it is suitable for hosting static websites directly from a GitHub repository. It also allows the website source files and version history to be managed in the same place.

## 3. Website Structure

The website contains the following main parts:

- A homepage named `index.html`
- A CSS file named `style.css`
- Links to Homework 1 files, including the PDF report, Markdown report, Python source code, system information, and running output
- A Homework 2 report written in Markdown
- A simple project log describing the construction process

## 4. Integration with Homework 1

The homepage includes links to the files produced in Homework 1:

- `report.pdf`
- `report.md`
- `matmul.py`
- `system_info.txt`
- `result_python.txt`

This integration allows visitors to access the previous assignment directly from the personal blog website.

## 5. Git Version Control Process

Git was used to manage the development process. The project was updated through several meaningful commits, including:

1. Adding Homework 1 files
2. Fixing Markdown formatting
3. Adding a basic blog stylesheet
4. Updating the homepage as a personal blog
5. Adding the Homework 2 Markdown report
6. Generating an HTML version of the Homework 2 report

These commits reflect the logical development process of the website.

## 6. Deployment Process

The website was deployed using GitHub Pages. The repository is named `lizihanFly.github.io`, and the website is published from the `main` branch and the root directory.

After pushing the files to GitHub, GitHub Pages automatically deployed the website. The final website can be accessed at:

https://lizihanfly.github.io/

## 7. Reflection

Through this assignment, I learned how to build a simple static website, organize project files, use Git for version control, and deploy a website through GitHub Pages. Compared with only writing local files, publishing the website online made the project easier to share and review.

This assignment also helped me understand the connection between HTML pages, Markdown documents, Git commits, and online deployment.
