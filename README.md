# Lizihan Software Technology Portfolio

Live website: <https://lizihanfly.github.io/>

This repository is a static GitHub Pages portfolio for four Software Technology assignments.

## Interactive Projects

- **Assignment 3, FileLens Agent Lab:** browser file analysis plus configurable online LLM and local Ollama adapters.
- **Assignment 4, Defend Your Thesis:** a Canvas survival game with character selection, waves, scoring, keyboard/mouse/touch controls, and local best scores.

## Main Files

- `index.html`: portfolio homepage
- `agent-lab.html` / `agent-lab.js`: Assignment 3 application
- `thesis-defense.html` / `thesis-defense.js`: Assignment 4 application
- `assignment3-report.md`: Assignment 3 documentation
- `assignment4-report.md`: Assignment 4 documentation
- `AI_DEVELOPMENT_LOG.md`: AI-assisted development record

No API keys or private credentials are stored in this repository.
